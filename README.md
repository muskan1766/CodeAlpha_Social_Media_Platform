# Social Media Platform

A full-stack social media web application that allows users to create accounts, share posts, interact with content, and connect with other users.

## Features

* User Registration
* User Login
* User Profile Management
* Follow System
* Create Posts
* Like Posts
* Comment on Posts
* Persistent Data Storage with MongoDB

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

## Project Structure

backend/

* models/
* server.js

frontend/

* css/
* js/
* register.html
* login.html
* profile.html
* index.html

## Installation

1. Install dependencies

npm install

2. Create a .env file in the backend folder and add:

MONGO_URI=your_mongodb_connection_string

3. Start the server

node server.js

4. Open register.html using Live Server.
